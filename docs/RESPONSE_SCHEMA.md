# Schema de Resposta — Darshan (7 fases)

## 7 fases, uma resposta por vez

Não são 7 linhas em uma única resposta. São **7 fases** do ritual: em cada interação o usuário recebe **uma mensagem** (da fase atual) e, ao continuar, avança para a próxima fase. Cada resposta aparece uma vez; novas interações podem mudar o conteúdo.

### As 7 fases

1. **Luz** — frase-oráculo  
2. **Pulso Jyotish** — qualidade do tempo agora (usa mapa do usuário se houver)  
3. **Arquétipo Chinês** — ciclo anual vigente  
4. **Elemento** — Ayurveda (Terra/Água/Fogo/Ar/Éter)  
5. **Consciência** — guna predominante (Sattva/Rajas/Tamas)  
6. **Ação mínima** — prática corporal segura (30–90s)  
7. **Pergunta final** — devolução à presença  

### Fluxo

- Usuário inicia → recebe mensagem da fase 1.  
- Usuário continua (ou responde) → recebe mensagem da fase 2.  
- Repete até a fase 7.  
- Após a fase 7, pode iniciar uma nova rodada (histórico limpo).
